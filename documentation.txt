# Agora AI - Complete Documentation

## 1. Overview and Workflow
Agora AI is a Multi-Agent Reasoning & Debate Engine. It orchestrates a structured, multi-round debate among several specialized AI agents to analyze complex topics, ethical dilemmas, or policy question.

The workflow operates across four distinct rounds:

- **Round 1 — Core Perspectives:**
  The Ethical Agent advocates for the affirmative or ethical standpoint. Following this, the Legal Agent presents an analysis from a legal and regulatory perspective, building upon the context given by the Ethical Agent.
- **Round 2 — Impact Perspectives:**
  The Economic Agent assesses the financial and economic implications. Next, the Social Agent presents its view highlighting social and community impacts, taking into consideration arguments from both Round 1 and the Economic Agent.
- **Round 3 — Refinement:**
  With all perspectives on the table, the Ethical Agent and Legal Agent refine and strengthen their original arguments, addressing concerns and nuances raised by the other agents.
- **Final Round — Consensus & Judgment:**
  The Consensus Agent reviews the entire debate history (including refined arguments) to deliver a definitive, balanced conclusion and recommendation.

*(Note: There is also an internal Clarifier Agent/moderator role designed for Situation Analysis mode to clarify vague scenario statements.)*

## 2. AI Models Used
The system natively supports local inference (via Ollama) and cloud inference (via Groq API) depending on availability. If a Groq API key is present, it will default to lightning-fast cloud models; otherwise, it will fall back to local models.

### Cloud Models (Groq API)
- **Ethical Agent:** Llama 3.3 70B Versatile
- **Legal Agent:** Llama 3.1 8B Instant
- **Economic Agent:** Llama 3.3 70B Versatile
- **Social Agent:** Llama 3.1 8B Instant
- **Consensus Agent:** Llama 3.3 70B Versatile
- **Clarifier Agent:** Llama 3.1 8B Instant

### Local Models (Ollama)
- **Ethical Agent:** Gemma 2 (2B)
- **Legal Agent:** Llama 3.2 (3B)
- **Economic Agent:** Qwen 2.5 (3B)
- **Social Agent:** Mistral (7B)
- **Consensus Agent:** Phi-3.5 Mini
- **Clarifier Agent:** Qwen 2.5 (3B)

## 3. How to Use the Website

### Prerequisites
1. Python 3.10+ installed on your machine
2. Optional: A Groq API Key (for high-speed cloud inference). If omitting, you must have Ollama installed and running locally with the models specified above.

### Setup Instructions
1. Install the required Python dependencies:
   `pip install -r requirements.txt`
2. **Configure API Key (Optional):**
   Copy `.env.example` to `.env` and set `GROQ_API_KEY=your_key_here`. 
   *(If you are using Ollama, ensure it is running in the background).*

### Running the Application (Two Parts)
**1. Start the Backend API (FastAPI):**
Open a new terminal and run:
`python -m uvicorn backend.main:app --reload --port 8080`
*(Make sure the port matches the `API_URL` configuration inside Streamlit if it's customized).*

**2. Start the Frontend Application (Streamlit):**
Open a second terminal window and run:
`streamlit run streamlit_app.py`

### Operating the Tool in Your Browser
Once the Streamlit interface opens in your browser (usually at `http://localhost:8501`), follow these steps:
1. **Enter a Topic:** Type your debate topic or dilemma into the text area. Alternatively, click one of the suggested example buttons (e.g., "UBI Worldwide?").
2. **Select the Mode:** Choose between "Debate Mode" for a classic formal argument or "Situation Analysis" for analyzing personal or professional dilemmas.
3. **Select Personality:** Adjust the overall tone of the debate (Balanced, Academic, Aggressive, Friendly, or Philosophical).
4. **Initiate:** Click the **"Start Debate Engine"** button.
5. **Observe:** Watch the progress as agents dynamically stream their arguments across four rounds, concluding with a beautifully formatted final consensus verdict.
